## 怎么用

**打印贴墙**(推荐): 用 PDF,A4 横向,缩放 100%。
**电子查阅**: PNG 适合塞进 GoodNotes、Notability、Obsidian Canvas、飞书文档。
**搜索**: PDF 内文字可被 macOS Spotlight / Everything 索引。

## 命名规范

```
<topic>-YYYY-MM.{pdf,png}
```

- `topic` 为 kebab-case,与 README 表格 anchor 对齐
- 月份后缀方便区分版本。
